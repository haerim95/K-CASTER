#### ↗️ 실제 접속이 가능한 사이트입니다.

## 개요

> `k-caster` 프로젝트는 종잡을 수 없는 요즘 날씨 탓에 옷을 고르는데 어려움이 많은 이들을 위한 **커뮤니티형** 웹 애플리케이션 입니다.  
> 내가 있는 지역을 선택하여 현재 날씨를 확인하고, 내 지역에 있는 사람들의 게시글을 보고 오늘의 날씨와 함께 알맞은 옷차림을 알 수 있습니다.

## 사용언어

##### Frontend

- Next.js
- React.js
- Redux
- Redux-saga
- Styled-Component

##### Backend

- Node.js
- express
- mySQL

## 날씨 API

[OpenWeather](https://openweathermap.org/) 오픈API 사용

## 배포환경

**AWS** 클라우드 서버를 이용한 배포
